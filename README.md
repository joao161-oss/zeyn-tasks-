# 📝 Zeyn Tasks

O **Zeyn Tasks** é uma aplicação de lista de tarefas (To-Do List) moderna e funcional, desenvolvida para ajudar na organização pessoal com foco em categorias e prazos.

---

## 🚀 Funcionalidades

* **Gestão de Tarefas:** Criação, edição e exclusão de deveres.
* **Categorização:** Organize tarefas por etiquetas como Escola, Casa, Lazer e Trabalho.
* **Controlo de Prazos:** Indicação visual de datas de vencimento e alerta de tarefas atrasadas.
* **Persistência de Dados:** As tarefas são salvas no navegador através do `localStorage`, para que não se percam ao fechar a página.
* **Filtros Inteligentes:** Visualize todas as tarefas, apenas as completas ou apenas as pendentes através da barra lateral.
* **Interface Moderna:** Design responsivo com suporte a Dark Mode, efeitos de Glassmorphism e animações interativas.
* **Drag and Drop:** Organize a ordem das suas tarefas arrastando e soltando os cards.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica da aplicação.
* **CSS3:** Estilização avançada com variáveis, gradientes e Flexbox/Grid.
* **JavaScript (Vanilla):** Lógica de manipulação do DOM, filtros e armazenamento local.
* **Font Awesome:** Ícones interativos.

---

## 💻 Como usar

1.  Crie uma tarefa na barra lateral direita, escolhendo uma categoria e uma data.
2.  Clique no título de uma tarefa para marcá-la como concluída.
3.  Utilize o ícone de lápis (✏️) para editar o nome, a data ou a categoria de um dever existente.
4.  Pesquise tarefas específicas utilizando a barra de busca com o ícone da bússola.
